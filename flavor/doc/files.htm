Flavor Package

Distribution Content

The Flavor software Version 5.1 comes with all the source code and tools to build/rebuild the executables and libraries. The code has been tested with Windows 98, NT, 2000 and XP, Linux, IBM AIX, SUN Solaris 8 and HP-UNIX. Also, for the Win32 platform including Windows 98, NT 2000 and XP, we include precompiled libraries, the executable of the Flavor translator, and bitgen (the bitstream generator) so that they can readily be used. For other platforms, the code can be compiled with minimal effort.

The Flavor Version 5.1 Package Includes

readme.html
The starting page of the Flavor documentation.
 
configure
A script for automatically detecting your system environment and generating Makefiles for the source code and samples. For more information on how to build the source code, see the Source Code page and for more information on how to build the samples, refer to the Samples page.
 
Makefile.in
A Makefile gets generated from this file by running the configure script. The Makefile, then, can be used to compile both the source code and the sample code in UNIX/Linux platforms.
 
COPYING.txt
The Flavor Artistic License. This covers the use and (re)distribution of this package.
 
src\flavorc
This directory contains the source code for the Flavor translator - flavorc.
 
src\lib\cpp
This directory contains the source code for the Flavor library - C++.
 
src\lib\java
This directory contains the source code for the Flavor library - Java.
 
src\lib\buffbitstream\cpp
This directory contains the source code for the Flavor library (for MP3 files) - C++.
 
src\lib\buffbitstream\java
This directory contains the source code for the Flavor library (for MP3 files) - Java.
 
src\bitgen
This directory contains the source code for the bitstream generator - bitgen.
 
doc
The documentation directory, including the Flavor specification and user's manual.
 
samples
Contains samples demonstrating the usage of Flavor. See the Samples page for more information about the provided samples.
 
samples\data
Contains sample data for testing the sample programs.
 
bin\flavorc.exe
Win32 binary of the Flavor translator. It takes as an input a .fl file and generates an equivalent set of .h and .cpp files, .java files, or a .xsd file.
 
The Flavor translator for UNIX/Linux platforms can be built by compiling the source in the src\flavorc directory. For more information about building the source refer to the Source Code page.
 
bin\bitgen.exe
Win32 binary of the bitstream generator. It takes as an input a .xml file, and optionally a .xsd file, and generates an equivalent bitstream. It uses Xerces-C++ Version 2.2.0 XML parser (xerces-c_2_2_0D.dll) to parse XML files.
 
The bitstream generator for UNIX/Linux platforms can be built by compiling the source in the src\bitgen directory. Since it utilizes the Xerces-C++ XML parser, the package must be downloaded from http://xml.apache.org and the necessary include files and library must be linked to properly compile bitgen. For more information, refer to the Source Code page.
 
include
Contains the same header files as the ones in the src\lib\cpp and src\lib\buffbitstream\cpp directories. They are automatically copied to the include directory as the C++ Flavor library is built. These files are included in the sample code.
 
lib\cpp\flavor.lib, flavormp3.lib
The flavor.lib file is the Flavor run-time library for C++ in the Win32 platform. This library provides support for bitstream I/O operations. Its API is very simple, and developers can use their own libraries as long as they expose the interface that flavorc expects. Such abstract interface is defined in flavori.h, and flavormp3.lib is another library (used for parsing MP3 files) derived from the interface. Note that the library is compiled using Microsoft Visual C++ 6.0; if you use a different compiler, you will need to rebuild the library as C++ name mangling is compiler-dependent. More information about the run-time library is provided in the user's manual.
 
The UNIX/Linux version of the C++ libraries described above can be built by compiling the source in the src\lib\cpp and src\lib\buffbitstream\cpp directories. For more information about building the source, refer to the Source Code section.
 
lib\java\flavor, flavormp3
Flavor run-time libraries for Java supporting bitstream I/O operations. More information about the run-time library is provided in the user's manual.
