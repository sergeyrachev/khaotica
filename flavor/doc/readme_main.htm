Welcome

Thank you for downloading Flavor Version 5.2. Flavor (which stands for "Formal Language for Audio-Visual Object Representation") is a multimedia description language designed as an extension of the typing system of C++ and Java so that it incorporates bitstream representation symantics. It allows the complete description of how multimedia information is laid out in a bitstream. For example, the bitstream structure of coded data such as GIF, JPEG and MPEG can be easily described using Flavor.

Flavor comes with a translator (flavorc) that reads Flavor source files - .fl files - and generates optimized C++ (or Java) code that implements the specified bitstream syntax. The generated code is placed in a pair of .h and .cpp (or a set of .java) files, and includes two methods: get() and put(). The former reads an object's information from a bitstream and places it into class member variables, while the latter writes this information to the bitstream. As a result, the programmer does not have to write any code for reading or writing multimedia data. In addition, the translator can automatically insert tracing code so that one can get a detailed dump of the information contained in a bitstream. This is invaluable during code development.

Since Version 5.0, the translator has been enhanced to also support XML. With the enhanced translator, in addition to the get() and put() methods, the generated C++/Java code can include a method (putxml()) for producing XML documents that correspond to the bitstreams described by Flavor. The description can also be used to generate a corresponding XML schema. Additionally, a software tool (bitgen) for converting XML representation of multimedia data back into bitstream form is provided. The enhanced translator and bitgen comprise XFlavor, a framework for providing XML features in media representation. Using XFlavor, multimedia data can be converted back and forth between the binary and XML representations.

This version of translator is incremental release and it is backward compatible with Versions 4.x, 3.x and 2.x with corrections to all the bugs that have been reported. Flavor is available for the Win32 and most of the UNIX/Linux platforms. The package includes the source code for the Flavor translator and the run-time library; thus, the binaries can be recompiled under any environment that has C/C++ and/or Java compiler. If you are familiar with the old versions, you may want to consult the What's New section.

Flavor has been used in the MPEG-4 standardization activity for the description of the bitstream syntax; it has been used as the syntax description language in the MPEG-4 Systems and Structured Audio specifications. You can find more information about MPEG-4 at the MPEG web site http://www.chiariglione.org/mpeg, or at the MPEG-4 Systems (a subgroup of MPEG-4) web site http://garuda.imag.fr/MPEG4.

For more information on the background of Flavor and a lot more, you can also visit the Flavor web site at http://flavor.sourceforge.net.
